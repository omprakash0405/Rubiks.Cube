## ✨ Features  

- 🔥 Double-tap to start/stop timer  
- 📊 Real-time statistics tracking  
- 🏆 Best time display  
- 📉 Worst time display  
- 📈 Average of 5, 12, and 25 solves  
- 🎨 Clean and distraction-free UI  
- ⚡ Fast and responsive performance  

---

## 🧠 How It Works  

1. Open the website  
2. Double-tap anywhere on the screen  
3. Timer starts  
4. Double-tap again to stop  
5. Your solve gets recorded in stats  

Perfect for speedcubers who want quick practice tracking.
